# My UI Library
This project is a collection of simply styled React UI components. The objective is to create components that look great as they are but can be easily restyled, including with Tailwind CSS.

---

### Check It Out!
Clone the repository
```bash
git clone https://github.com/michaelcherrera84/My-UI-Library.git
```
Install dependencies
```bash
cd my-ui-lib && npm install
```
Check out the Storybook
```bash
npm run storybook
```
